## Cara pakai

1. Salin file ini ke server atau jalankan pada mesin lokal.
2. `npm install`
3. Buat `.env` berdasarkan `.env.example` dan isi `OPENAI_API_KEY` (server-side!)
4. `npm start`
5. Buka `http://localhost:3000`

Catatan: jangan taruh API key di frontend. Biaya penggunaan OpenAI berlaku.
